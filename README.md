# cellular-automata

Dependencies:  
*sdl2*  
*sdl2_image*  

#### controls:
"-" to reduce brush size and "=" to increase  
spacebar to pause  
all mouse buttons draw  

In lines 37-54 of main.c you can find code to change the type of life.
